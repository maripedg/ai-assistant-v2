Component fm_subscription
Visibility public
Transaction required

Description

1 int    PURCHASE_START_UNIT In Out

The start date can be specified in relative units if it is not set for First Usage or Immediate or Absolute. The values can be Seconds(1) Minutes(2) Hours(3) Days(4) Months(5) Act Cycles(8)
