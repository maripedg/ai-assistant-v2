PCM_OP_SUBSCRIPTION_READ_ACCT_PRODUCTS

Component fm_subscription
Visibility public
Transaction supports

Description

PCM_OP_SUBSCRIPTION_READ_ACCT_PRODUCTS input and output flist specifications. The opcode returns the purchased offerings of an account, optionally even the service addresses, item products and/or cancelled offerings, in a hierarchical service->deal->offering structure.

Flags to indicate whether item offerings and/or cancelled offerings need to be fetched. If not used, only the purchased active and inactive subscription offerings are fetched. PIN_CUST_RAP_FETCH_ITEM_PRODUCTS (0x01) Get item products also PIN_CUST_RAP_FETCH_DELETED_PRODUCTS (0x02) Get cancelled offerings also PIN_CUST_RAP_FETCH_BASE_PRODUCTS_ONLY (0x04) Get only base subscribed products PIN_CUST_CASE_INSENSITIVE (0x10) For case insensitive search PIN_CUST_INCLUDE_SUB_SERVICE (0x20) To include sub services in the search PIN_CUST_INCLUDE_SUB_DEVICE (0x40) To include contact information in the search PIN_CUST_INCLUDE_CONTACT_INFO (0x80) To include contact information.
