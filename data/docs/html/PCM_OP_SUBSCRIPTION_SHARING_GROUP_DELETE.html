PCM_OP_SUBSCRIPTION_SHARING_GROUP_DELETE

Component fm_subscription
Visibility public
Transaction required

Description

PCM_OP_SUBSCRIPTION_SHARING_GROUP_DELETE input and output flist specifications. This opcode deletes either: 1) the sharing group object, if no MEMBERS/DISCOUNTS/SPONSORS/PROFILES/PRODUCTS array is passed, or 2) the input MEMBERS/DISCOUNT/SPONSORS/PROFILES/PRODUCTS array.
